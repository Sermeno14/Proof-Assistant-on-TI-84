# Proof-Assistant-on-TI-84
This project aims to make a proof assistant for the TI-84 SE.

`tester.8xp` Is a save of stage 1, where we (finally) got the input to corresponding with the proper character on screen.
`tester2.8xp` Is a save of stage 2, getting the input to check against different flags and jump to different routines in the code.
